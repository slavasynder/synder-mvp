How to export the file from Xero

  1. Xero → Reporting → Account Transactions.
  2. In the Account filter, tick all four: the clearing account, Accounts Receivable, Accounts Payable, Realized Currency Gains.
  3. Grouping / Summarise = None. Choose the date range.
  4. Add columns: Date, Source, Contact, Description, Invoice Number, Reference, Net, Currency, Account. Also keep Net (USD) if the org has FX activity.
  5. Update → Export as Excel.

One reconciliation run = one clearing account. If you have several clearing accounts, run the export (and the normalizer) once per clearing account.
